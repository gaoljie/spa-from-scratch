## SPA-FROM-SCRATCH

This is a guideline to build an modern SPA project using React framework step by step.

We use following framework and packages:

- UI Framework - React
- Formatter - eslint, prettier, tslint(for typescript repo)
- Bundler - Webpack
- Style - Styled Components
- Router - Reach Router
- Flux - Mobx
